Template for UNSW-themed Quarto slides

Author

Dr Patrick Laub

This is a template for using Quarto to create reveal.js HTML slides according to a UNSW theme. You can download the template as a zip file or you can clone the Github repository.

I recommend using RStudio to render the main template.qmd file. Or, if you prefer the commandline, run quarto render template.qmd. In either case, you’ll need to have Quarto installed first. This is the generated slides.

Note

Reveal.js slides can be progressed by pressing the spacebar key, or by clicking on the arrows at the edge of the slide.
